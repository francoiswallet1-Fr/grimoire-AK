# Grimoire AK Interactive 🎨

Application web de recettes de peinture AK Interactive, installable sur mobile.

## Déploiement sur GitHub Pages (5 minutes)

### 1. Créer un compte GitHub
Rendez-vous sur [github.com](https://github.com) et créez un compte gratuit.

### 2. Créer un nouveau dépôt
- Cliquez sur **"New repository"**
- Nom : `grimoire-ak` (ou ce que vous voulez)
- Visibilité : **Public** (obligatoire pour GitHub Pages gratuit)
- Cliquez **"Create repository"**

### 3. Uploader les fichiers
- Cliquez **"uploading an existing file"**
- Glissez-déposez les 5 fichiers de ce dossier :
  - `index.html`
  - `manifest.json`
  - `sw.js`
  - `icon-192.png`
  - `icon-512.png`
- Cliquez **"Commit changes"**

### 4. Activer GitHub Pages
- Allez dans **Settings** → **Pages**
- Source : **Deploy from a branch**
- Branch : **main** / **root**
- Cliquez **Save**

### 5. Votre URL
Après 1-2 minutes, votre grimoire sera disponible à :
`https://VOTRE_NOM.github.io/grimoire-ak/`

## Installation sur téléphone

### Android (Chrome)
1. Ouvrez l'URL dans Chrome
2. Un bandeau **"Installer"** apparaît automatiquement
3. Ou : menu ⋮ → **"Ajouter à l'écran d'accueil"**

### iPhone (Safari)
1. Ouvrez l'URL dans Safari
2. Bouton **Partager** → **"Sur l'écran d'accueil"**
3. Tapez **Ajouter**

Une fois installé : fonctionne **hors ligne**, plein écran, icône sur l'accueil.
